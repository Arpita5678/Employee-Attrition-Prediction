# Employee Attrition Dashboard

## 📌 Project Overview
This project analyzes employee data to identify patterns of attrition and burnout. The goal is to help HR teams detect early warning signs and take proactive actions.


## 🎯 Objective
To build an interactive dashboard that:
- Identifies employees at risk of leaving
- Highlights key factors like overtime, job satisfaction, and department trends
- Supports data-driven HR decisions

## 📊 Tools Used
- Power BI
- Excel

## 📁 Dataset
The dataset includes employee-related information such as:
- Age, Gender, Department
- Job Satisfaction, Environment Satisfaction
- Monthly Income, Overtime
- Attrition (Yes/No)

## 📈 Key Insights
- Employees working overtime show higher attrition risk  
- Job satisfaction directly impacts employee retention  
- Attrition varies across departments  


## 📊 Dashboard Features
- Attrition Count (KPI)
- Attrition by Overtime (Pie Chart)
- Attrition by Department (Bar Chart)
- Attrition by Job Satisfaction (Bar Chart)



## 🚀 Outcome
This dashboard helps organizations move from reactive to proactive HR strategies by identifying at-risk employees early.
